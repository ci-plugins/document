## 数据返回包装模型List项目-显示模型

| 参数名称 | 参数类型 | 必须 | 参数说明 |
| :--- | :--- | :--- | :--- |
| result | boolean | 否 | 请求结果 |
| code | integer | 是 | 状态码 |
| data | List&lt;[项目-显示模型](query-all-items.md)&gt; | 否 | 数据 |
